# Aero-Bridge

## Overview

Aero-Bridge is a UAV-assisted emergency mesh communication network designed for disaster-response scenarios where conventional communication infrastructure is unavailable.

The system uses ESP32 microcontrollers and NRF24L01 wireless transceivers to establish a lightweight mesh network capable of relaying emergency messages between multiple nodes. A command centre dashboard receives and monitors incoming alerts in real time.

## Features

* ESP32-based field nodes
* NRF24L01 mesh communication
* Captive portal for emergency message submission
* Multi-hop packet forwarding
* TTL-based routing
* Duplicate packet suppression
* Heartbeat-based node discovery
* Command centre monitoring dashboard

## Hardware Used

* ESP32
* NRF24L01
* MPU6050
* BMP280

## Applications

* Disaster response
* Search and rescue operations
* Emergency communication
* UAV-assisted networking
